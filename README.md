> **YTDOWN** is a **Flask web application** that lets you download YouTube videos and audio directly from your browser ‚ÄĒ no desktop app needed. Just paste a URL, pick your format and quality, and hit download. Powered by `pytubefix` + `FFmpeg` for high-quality merging.

---

## ‚ö° What Makes This Different

| Feature | YTDOWN | Other Tools |
|--------|--------|-------------|
| ūüĆź Web-based (Flask) | ‚úÖ Yes | ‚ĚĆ Usually desktop-only |
| ūüíĺ No server storage | ‚úÖ Streams to buffer | ‚ĚĆ Saves to server disk |
| ūüé¨ Auto video+audio merge | ‚úÖ FFmpeg powered | ‚ĚĆ Separate files |
| ūüöę Zero ads | ‚úÖ Always | ‚ĚĆ Usually full of ads |
| ūüĒß Self-hostable | ‚úÖ Yes | ‚ĚĆ Locked to their server |

## ūüöÄ Getting Started

### Prerequisites

```bash
# Check Python version (3.8+ required)
python --version

# Check FFmpeg (required for 1080p+ merging)
ffmpeg -version
```

> ūüď• FFmpeg not installed? Get it at [ffmpeg.org/download](https://ffmpeg.org/download.html) and add to PATH.

---

### Installation

**Clone the repo**
```bash
git clone https://github.com/SOMU3103/YTDOWN.git
cd YTDOWN
```

**Create & activate virtual environment**
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

**Install dependencies**
```bash
pip install -r requirements.txt
```

Or install manually:
```bash
pip install flask pytubefix
```

**Run the Flask server**
```bash
python app.py
```

Open your browser at:
```
http://127.0.0.1:5000
```

## ūüéģ Usage

```
1.  Open http://127.0.0.1:5000 in your browser
2.  Paste any YouTube URL
3.  Click "Fetch" ‚ÄĒ qualities are loaded dynamically
4.  Select:  [ MP4 Video ]  or  [ MP3 Audio ]
5.  Choose resolution  (360p / 480p / 720p / 1080p / 4K)
6.  Hit Download ‚ÄĒ file saves directly to your device
```

## ‚öôÔłŹ API Reference

### `POST /get_qualities`

Fetch all available streams for a YouTube URL.

**Request**
```json
{
  "url": "https://www.youtube.com/watch?v=XXXXXXXXXXX"
}
```

**Response**
```json
{
  "success": true,
  "title": "Video Title Here",
  "duration": 243,
  "video_qualities": [
    { "resolution": "1080", "label": "1080p", "fps": 30 },
    { "resolution": "720",  "label": "720p",  "fps": 30 }
  ],
  "audio_qualities": [
    { "bitrate": "128kbps", "label": "128kbps" }
  ]
}
```
